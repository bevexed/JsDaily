# DOM
* Document OF Model
## 节点 `node`
* 元素节点 `element node`
* 文本节点 `text node`
* 属性节点 `attribute node`
* 元素的所以子元素 `childNodes`
  * nodeType
  1. 元素节点的nodeType是1
  2. 属性节点的nodeType是2
  3. 文本节点的nodeType是3
  * nodeValue
  * firstChild
  * lastChild
